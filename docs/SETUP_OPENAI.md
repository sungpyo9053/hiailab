# 🟢 OpenAI API 키 설정하기

HiaiLab의 "뽑기" 결과를 진짜 AI가 만들도록 하려면 OpenAI API 키가 필요합니다.

> **키가 없어도 HiaiLab은 동작합니다.** (MOCK 결과로 나옵니다.)
> 천천히 발급하셔도 됩니다.

---

## 1. API 키가 뭔가요?

쉽게 말해 **"AI 회사에 본인 계정으로 결제하면서 쓰겠다고 신청한 비밀번호"** 입니다.

- `sk-` 로 시작하는 긴 글자입니다. (예: `sk-abc123...`)
- **유출되면 다른 사람이 본인 카드로 AI를 쓰게 됩니다.** 절대 GitHub, 블로그, 카톡, 스크린샷 등에 노출되면 안 됩니다.
- 유출되면 OpenAI 대시보드에서 즉시 **폐기(Revoke)** 하고 새 키를 발급하세요.

---

## 2. 발급 방법 (5분)

1. <https://platform.openai.com/> 접속 → 로그인 (구글 계정 가능)
2. 우측 상단 프로필 아이콘 → **View API keys**
   - 직링크: <https://platform.openai.com/api-keys>
3. **`+ Create new secret key`** 클릭
4. 이름 입력 (예: `hiailab-local`) → **Create secret key**
5. 화면에 `sk-...` 로 시작하는 키가 한 번만 보입니다. **복사해서 안전한 곳에 보관**하세요.
   - 창을 닫으면 다시 볼 수 없습니다. 잃어버리면 새로 발급해야 합니다.

> 처음 가입한 경우 결제 수단(카드) 등록이 필요할 수 있습니다.
> 자판기 한 번 뽑는 비용은 보통 **1원 미만** 입니다. (gpt-4o-mini 기준)

---

## 3. HiaiLab에 키 넣는 방법

### 방법 A — `/setup` 페이지에서 (가장 쉬움)

1. <http://localhost:3000/setup> 접속
2. "1) OpenAI 설정" 칸에 키 붙여넣기
3. **저장** 클릭 → 상단 배지가 `AI REAL` (초록) 로 바뀜
4. **연결 테스트** → `✓ OpenAI 연결 OK` 가 나오면 끝

키는 입력 즉시 **이 컴퓨터의 `.hiailab/config.enc.json`** 파일에 AES-256-GCM 암호화되어 저장됩니다. 입력 칸은 비워지고, 다시 화면에 원문이 보이지 않습니다.

### 방법 B — `.env.local` 파일에 직접

```env
OPENAI_API_KEY=sk-abc123...
```

저장 후 dev 서버를 재시작 (`Ctrl + C` → `npm run dev` 다시).

> `.env.local` 은 자동으로 `.gitignore` 에 등록되어 있어 GitHub로 새지 않습니다.

---

## 4. 키가 동작하는지 확인하는 가장 빠른 방법

`/setup` 페이지 → "OpenAI 설정" 박스 → **연결 테스트** 버튼.

- `✓ OpenAI 연결 OK` → 정상
- `✗ OpenAI 키가 거부되었습니다 (401)` → 키가 틀렸거나 폐기됨. 다시 발급해서 저장.

---

## 5. 비용은?

HiaiLab은 기본적으로 **`gpt-4o-mini`** 모델을 사용합니다.

- 답장 자판기 1회: 보통 **₩1 미만**
- 회의록 자판기 1회 (긴 회의록): **₩5 ~ 30** 정도
- 카피 자판기 1회: **₩2 ~ 10** 정도

> 정확한 요금은 OpenAI 대시보드의 "Usage" 탭에서 확인하실 수 있습니다.

---

## 6. 키 유출 시 대응

1. OpenAI 콘솔의 API keys 페이지에서 해당 키 **Revoke** (즉시 무효화)
2. 새 키 발급
3. `/setup` 페이지에서 새 키로 저장
4. 만약 Git에 실수로 커밋했다면, 커밋을 되돌리는 것만으로는 부족합니다. 반드시 **Revoke 먼저** 하세요.

---

## 7. ⚠️ 절대 하면 안 되는 것

- ❌ 키를 코드 파일에 하드코딩 (`const key = "sk-..."`)
- ❌ 키를 그대로 GitHub에 push
- ❌ 키를 카톡/슬랙/이메일로 공유
- ❌ 스크린샷에 키가 보이게 캡처
- ❌ 다른 사람에게 본인 키를 빌려주기
