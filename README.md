# 🥤 오토벤딩 (AutoVending)

> 필요한 AI 자동화를 **자판기처럼 바로 뽑아 쓰는** self-hosted 서비스

회사 메일에 답장 쓰기, 회의록 정리하기, 상품 카피 만들기 같은 반복 업무를 **카드 한 번 누르면 결과가 나오는 자판기**로 묶었습니다. 코드는 GitHub에서 받아 **본인 컴퓨터 또는 본인 개인 서버에 직접 올려서** 사용합니다. 입력한 키와 결과는 어디로도 전송되지 않습니다.

## 1. 한 줄 설명

> 자판기처럼 보이는 화면에서 카드를 누르고, 텍스트만 붙여넣으면, AI가 결과를 만들어서 **이메일이나 카카오톡으로 바로 보내주는** 1인용 자동화 도구.

---

## 2. 현재 되는 기능

| 자판기 | 입력 | 결과 |
| --- | --- | --- |
| ✉️ **답장 자판기** | 받은 이메일 본문 | 정중한 답장 4종 + 피해야 할 표현 |
| 📝 **회의록 자판기** | 회의 메모/녹취 | 요약 · 결정사항 · 담당자별 할 일 · 리스크 · 공유용 회의록 |
| 🛍️ **카피 자판기** | 상품명 + 장점 | 짧은 광고 / 상세페이지 / 긴 설득 카피 3종 |

추가로:
- **OpenAI 키 없이도** mock 결과로 끝까지 체험 가능
- **이메일(SMTP) 또는 카카오톡 "나에게 보내기"** 로 결과 바로 전송
- **`/setup` 페이지** — 키/토큰/SMTP를 브라우저에서 입력 (암호화 저장)
- **Docker 한 줄**로 개인 서버 배포

---

## 3. 사용 예시

```
1. 메인 페이지에서 "답장 자판기" 카드 클릭
2. 받은 메일 본문 붙여넣기
3. [₩900으로 뽑기] 클릭   ← POC라 실제 결제는 없습니다
4. AI 답장 결과 확인
5. [✉️ 내 이메일로 보내기] 또는 [💬 카카오톡 나에게 보내기]
```

## 4. 설치 방법

가장 빠른 방법은 **[QUICKSTART.md](./QUICKSTART.md)** 한 페이지를 따라 하는 것입니다. (10분 이내)

```bash
# 1) 코드 받기 (Git 사용)
git clone https://github.com/sungpyo9053/autovending.git
cd autovending

# 2) 환경변수 파일 만들기
cp .env.local.example .env.local
# .env.local 파일을 텍스트 에디터로 열어서 APP_ENCRYPTION_KEY 1줄만 채워도 됩니다.

# 3) 의존성 설치 + 실행
npm install
npm run dev
```

브라우저에서 <http://localhost:3000> → **`🛠️ 설정`** 버튼 → OpenAI 키 / SMTP / 카카오 토큰을 화면에서 저장.

> Git을 모르시면 GitHub 페이지 우측 상단의 녹색 **`<> Code`** 버튼 → **Download ZIP** 으로 받아도 됩니다. QUICKSTART에 안내가 있습니다.

## 7. 보안 주의사항 ⚠️

오토벤딩은 **Public GitHub에 그대로 올려도 안전한 구조**입니다. 단, 아래 규칙을 지켜주세요.

1. **실제 API 키, SMTP 비밀번호, 카카오 토큰은 GitHub에 절대 올리지 마세요.**
2. 키와 비밀번호는 `.env.local` 또는 `/setup` 페이지로만 입력하세요. 두 곳 모두 Git 커밋에서 자동 제외됩니다.
3. `/setup`에서 저장한 값은 **이 컴퓨터의 `.autovending/config.enc.json`** 에 AES-256-GCM 암호화로 보관됩니다. 외부로 나가지 않습니다.
4. 만약 실수로 키를 커밋했다면, 즉시 OpenAI/카카오/Gmail에서 해당 키를 **폐기·재발급**하세요. Git 히스토리에 남은 키는 회수가 안 됩니다.
5. 모든 system prompt는 `lib/prompts.ts`에 `import "server-only"`로 격리되어 클라이언트 번들에 포함되지 않습니다.

자세한 FAQ는 → [docs/FAQ.md](./docs/FAQ.md)

## 8. 현재 한계 (POC 단계)

- Gmail 받은편지함 **자동 읽기는 없습니다**. (이번 단계는 발송 전용)
- 카카오톡 **친구/고객 발송은 없습니다**. ("나에게 보내기" 만)
- 카카오 OAuth 자동 로그인은 없습니다. (토큰 수동 발급)
- 실제 결제는 붙어 있지 않습니다. (₩900은 자판기 컨셉용 표시)
